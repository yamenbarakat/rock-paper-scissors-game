# Rock Paper Scissors Game

An interactive Rock Paper Scissors game built with React and Sass.
This project is based on the Frontend Mentor challenge and includes a clean, responsive interface where the user plays against the computer.

## Overview

The app lets the player choose between rock, paper, and scissors, then generates a random computer choice and shows the round result (win, lose, or draw). It also tracks the score and supports a smooth experience on both mobile and desktop layouts.

## Features

- Play Rock, Paper, Scissors against the computer
- Random computer pick for each round
- Round result display (win/lose/draw)
- Score tracking
- Responsive design for different screen sizes
- Hover and focus states for interactive elements

## Getting Started

1. Install dependencies:
   `npm install`
2. Start the development server:
   `npm run dev`
3. Build for production:
   `npm run build`
4. Preview the production build:
   `npm run preview`
